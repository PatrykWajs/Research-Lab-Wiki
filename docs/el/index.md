# Research Lab Wiki

Αυτοκατευθυνόμενες, πλήρως τεκμηριωμένες έρευνες — κάθε μία ανιχνεύεται σε μελέτες με κριτική αξιολόγηση και επίσημες πηγές, ζυγισμένες ειλικρινά, και κλείνουν με εκτίμηση με βαθμολογία αποδεικτικών στοιχείων.

<div class="grid cards" markdown>

-   :material-flask-outline:{ .lg .middle } __BPA στις Θερμικές Αποδείξεις & Ανθρώπινη Υγεία__

    ---

    Πώς η δισφαινόλη Α από αποδείξεις ταμειακών και καρτών εισέρχεται στο σώμα, τι κάνει, και πόσο πραγματικός είναι ο κίνδυνος — για τον μέσο άνθρωπο έναντι των ταμείων. **29 τεκμηριωμένες μελέτες**, με βαθμολογία αποδεικτικών στοιχείων.

    [:octicons-arrow-right-24: Read the research](bpa-thermikes-apodeixeis.md)

-   :material-bone:{ .lg .middle } __Κολλαγόνο: Τι Πραγματικά Κάνει έναντι Τι Πιστεύουν οι Άνθρωποι__

    ---

    Ύπνος, έντερο, δέρμα, αρθρώσεις, οστά, μύες — ποιες αξιώσεις για το κολλαγόνο είναι πραγματικές, και ποιες είναι απλώς ένα αποτέλεσμα γλυκίνης. **74 τεκμηριωμένες μελέτες**, με σημαία χρηματοδότησης.

    [:octicons-arrow-right-24: Read the research](kollagono.md)

-   :material-arm-flex:{ .lg .middle } __Τεστοστερόνη: Τι Πραγματικά την Αυξάνει έναντι Τι Πιστεύουν οι Άνθρωποι__

    ---

    Διατροφή, ύπνος, συμπληρώματα, no-fap, έλαια σπόρων, ήλιος, πλαστικά — ποιες αξιώσεις "αυξάνουν την τεστοστερόνη φυσικά" ισχύουν, και ποιες (σόγια, ρόδι, no-fap, ήλιος-στους-όρχεις) είναι μύθοι. **96 τεκμηριωμένες μελέτες**, με βαθμολογία αποδεικτικών στοιχείων.

    [:octicons-arrow-right-24: Read the research](testosteroni.md)

-   :material-bottle-tonic-outline:{ .lg .middle } __Έλαια Σπόρων: Τι Πραγματικά Κάνουν έναντι Τι Πιστεύουν οι Άνθρωποι__

    ---

    Φλεγμονή, καρδιακή νόσος, παχυσαρκία, "τοξικό" λάδι τηγανίσματος — ποιες αξιώσεις "τα έλαια σπόρων είναι δηλητήριο" ισχύουν, και ποιες (ο μηχανισμός φλεγμονής, "προκαλούν καρκίνο," "αντικατάσταση με λίπος") είναι μύθοι ή μάρκετινγκ. **96 τεκμηριωμένες πηγές**, όλες οι PMID επαληθευμένες.

    [:octicons-arrow-right-24: Read the research](sporelaia.md)

-   :material-dumbbell:{ .lg .middle } __Κρεατίνη: Τι Πραγματικά Κάνει έναντι Τι Πιστεύουν οι Άνθρωποι__

    ---

    Βλάβη νεφρών, απώλεια μαλλιών, "είναι στεροειδές," φούσκωμα, πρέπει-να-κυκλώνεται — ποιες φόβοι για την κρεατίνη ισχύουν (σχεδόν κανένας), και τι πραγματικά κάνει για τους μύες, τον εγκέφαλο και την υγεία. Το πιο μελετημένο, καλύτερα τεκμηριωμένο, ασφαλέστερο αθλητικό συμπλήρωμα. **169 τεκμηριωμένες πηγές**, όλες οι PMID επαληθευμένες.

    [:octicons-arrow-right-24: Read the research](kreatini.md)

-   :material-scale-balance:{ .lg .middle } __Μεθοδολογία__

    ---

    Πώς κάθε έρευνα εδώ βαθμολογεί τα αποδεικτικά στοιχεία, δηλώνει εμπιστοσύνη, και επαληθεύει κάθε αναφορά πριν δημοσιευθεί.

    [:octicons-arrow-right-24: How this is graded](methodologia.md)
