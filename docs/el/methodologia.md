# Μεθοδολογία

Κάθε έρευνα εδώ ακολουθεί τους ίδιους κανόνες: αναφέρετε κάθε ισχυρισμό σε μια πρωτογενή πηγή, βαθμολογήστε τη δύναμη αυτής της πηγής, δηλώστε την εμπιστοσύνη ειλικρινά και αναφέρετε γεγονότα και αβεβαιότητα αντί για μια τακτοποιημένη ετυμηγορία. Τα ευρήματα στη συνέχεια **παρουσιάζονται καταταγμένα ανάλογα με το πόσο ισχυρά τα υποστηρίζουν οι αποδείξεις** — πρώτα η καθιερωμένη, πολυμελέτη συναίνεση· τελευταίοι οι ισχυρισμοί μιας μελέτης ή αμφισβητούμενοι — ώστε ο αναγνώστης να μπορεί να δει με μια ματιά πόσο βάρος αξίζει κάθε σημείο. Αυτή η σελίδα είναι το κοινό κλειδί στο οποίο αναφέρονται αυτές οι αναφορές.

## Επίπεδα εμπιστοσύνης

Κάθε αναφορά δηλώνει μια εμπιστοσύνη ανά ερώτηση — και συχνά μια διαφορετική εμπιστοσύνη για διαφορετικά μέρη της ίδιας ερώτησης (η έκθεση μπορεί να είναι βέβαιη ενώ η αιτιότητα της νόσου όχι).

| Επίπεδο | Τι σημαίνει |
|---|---|
| **Υψηλό** | Πολλαπλές ανεξάρτητες μελέτες συμφωνούν, ιδανικά με ελεγχόμενο ή ποσοτικοποιημένο σχεδιασμό (π.χ. μια δοκιμή έκθεσης με γάντια). Απίθανο να ανατραπεί με νέα δεδομένα. |
| **Μέτριο** | Ένα πραγματικό, συνεπές σήμα, αλλά οι αποδείξεις είναι κυρίως παρατηρητικές / διατομεακές, ετερογενείς ή πραγματικά αμφισβητούμενες μεταξύ ειδικών. Η κατεύθυνση είναι πιο σαφής από το μέγεθος. |
| **Χαμηλό–Μέτριο** | Πιθανό και συνεπές με τον μηχανισμό, αλλά όχι άμεσα αποδεδειγμένο για τον συγκεκριμένο ισχυρισμό — συμπεραίνεται από παρακείμενες αποδείξεις. Αντιμετωπίστε το ως υπόθεση, όχι ως γεγονός. |

## Βαθμοί αποδείξεων

Κάθε πηγή σε μια βιβλιογραφία φέρει έναν από αυτούς τους βαθμούς, ώστε ο αναγνώστης να μπορεί να δει με μια ματιά πόσο βάρος αξίζει ένας ισχυρισμός.

| Βαθμός | Τύπος πηγής | Βάρος |
|---|---|---|
| **Οργανισμός** | Επίσημη εκτίμηση κινδύνου ή κανονισμός (EFSA, US FDA, ECHA, NTP, EU νόμος). | Υψηλότερο για πολιτική / όρια έκθεσης· οι οργανισμοί μπορούν ακόμα να διαφωνούν — όταν το κάνουν, η αναφορά το λέει. |
| **Συστηματική ανασκόπηση / μετα-ανάλυση** | Συγκεντρωτική, προεγγεγραμμένη σύνθεση από πολλές πρωτογενείς μελέτες. | Υψηλότερη βαθμίδα πρωτογενούς απόδειξης για ένα αποτέλεσμα. |
| **Πρωτογενής ανθρώπινη** | Βιοπαρακολούθηση ή ελεγχόμενη μελέτη ανθρώπινης έκθεσης. | Ισχυρή για έκθεση· για νόσο, ασθενέστερη αν είναι διατομεακή παρά αν είναι προοπτική. |
| **Δευτερογενής** | Αφηγηματική ανασκόπηση, σχόλιο ειδικού ή μαρτυρία ειδικού. | Πλαίσιο και επιβεβαίωση, όχι ένα αυτόνομο γεγονός. |
| **Ειδήσεις** | Δημοσιογραφία. | Μόνο προσανατολισμός — ποτέ η μοναδική υποστήριξη για έναν ισχυρισμό. |

## Πώς επαληθεύεται κάθε αναφορά

Τα αυτοματοποιημένα εργαλεία "βαθιάς έρευνας" είναι χρήσιμα για την *εύρεση* πηγών αλλά μπορεί να αποτύχουν όταν προσπαθούν να αυτοεπαληθευτούν. Η ροή που χρησιμοποιείται εδώ αντιμετωπίζει την έξοδό τους ως οδηγούς και ελέγχει κάθε έναν με το χέρι:

1. **Συλλογή οδηγών.** Συγκεντρώστε υποψήφιους ισχυρισμούς και πηγές από αναζητήσεις — αντιμετωπίστε τους ως μη επαληθευμένους.
2. **Επαλήθευση έναντι της πρωτογενούς πηγής.** Ανοίξτε τη σελίδα DOI / PMID / οργανισμού κάθε μελέτης και επιβεβαιώστε τον ακριβή αριθμό ή απόσπασμα πριν δημοσιευτεί. Αν δεν μπορεί να επιβεβαιωθεί, παραλείπεται — ποτέ δεν υποτίθεται.
3. **Συμπλήρωση κενών.** Κάθε λεπτό μέρος της ερώτησης λαμβάνει νέες αναζητήσεις και πρωτογενή ανάγνωση, κάθε νέα πηγή αναφέρεται.
4. **Βαθμολόγηση και συγγραφή.** Μόνο τότε γράφεται η αναφορά, με κάθε ισχυρισμό δεμένο σε μια βαθμολογημένη πηγή.

## Η επανεπαλήθευση

Οι αναφορές δεν επαληθεύονται μία φορά και εμπιστεύονται για πάντα. Κάθε δημοσιευμένη αναφορά διαβάζεται περιοδικά **ξανά από τις πρωτογενείς πηγές της**, και αυτή η επανεπαλήθευση έχει τους δικούς της αυστηρούς κανόνες — επειδή τα αυτοματοποιημένα εργαλεία και ακόμη και οι προσεκτικές σημειώσεις κάνουν δύο συγκεκριμένα, επαναλαμβανόμενα λάθη.

1. **Κάθε αναγνωριστικό αντιστοιχείται με τον τίτλο στη βάση δεδομένων πηγών.** Για κάθε αναφερόμενο PMID, το ζωντανό PubMed αρχείο ανακτάται και ο τίτλος/συγγραφέας του ελέγχεται έναντι αυτού που ισχυρίζεται η αναφορά — επειδή ένα αναγνωριστικό μελέτης μπορεί σιωπηλά να δείχνει στο *λάθος έγγραφο* (μια εντελώς διαφορετική μελέτη). Αυτό έχει εντοπίσει πραγματικά λάθη: σε μια επανεπαλήθευση, δύο αναφορές αποδείχθηκαν ότι φέρουν το αναγνωριστικό ενός άσχετου εγγράφου και διορθώθηκαν.
2. **Κάθε αναφερόμενη μελέτη διαβάζεται από την περίληψή της**, όχι μόνο οι κρίσιμες — ώστε ένας αριθμός, μια κατεύθυνση αποτελέσματος ή μια χρηματοδοτική σύνδεση να μπορεί να διορθωθεί έναντι αυτού που πραγματικά αναφέρει η μελέτη.
3. **Οι διορθώσεις εμφανίζονται, δεν κρύβονται.** Όταν η επανεπαλήθευση αλλάζει κάτι, η αναφορά φέρει μια σύντομη σημείωση **"Τι άλλαξε"** που απαριθμεί κάθε διόρθωση (ένα λάθος αναγνωριστικό, ένα αντιστραμμένο εύρημα, ένα όνομα συγγραφέα), ώστε ο αναγνώστης να μπορεί να δει τη διόρθωση αντί για μια σιωπηλά επεξεργασμένη σελίδα.

Ο στόχος είναι ότι οι αποδείξεις ενός ισχυρισμού είναι ακριβώς τόσο ισχυρές όσο φαίνονται — και όταν δεν είναι, η σελίδα το λέει.

!!! note "Μόνιμοι κανόνες"
    - **Ποτέ μην επινοείτε** μια μελέτη, συγγραφέα, DOI ή αριθμό. Λείπει ή δεν μπορεί να επαληθευτεί → παραλείπεται, δεν συμπληρώνεται.
    - **Αναφέρετε τα πάντα.** Κάθε ισχυρισμός ανιχνεύεται σε μια αριθμημένη πηγή.
    - **Γεγονότα, όχι ετυμηγορίες.** Οι αμφισβητούμενοι πόντοι λαμβάνουν τις αποδείξεις και από τις δύο πλευρές και μια σαφή ένδειξη του τι είναι πραγματικά αβέβαιο — τα αποτελέσματα που χρηματοδοτούνται από τη βιομηχανία και τα μηδενικά αποτελέσματα ζυγίζονται, δεν αγνοούνται, και η θέση του τύπου ενός οργανισμού ποτέ δεν υπερβαίνει τις πρωτογενείς αποδείξεις.
    - **Ημερομηνία τρέχουσας βιβλιογραφίας.** Κάθε αναφορά σφραγίζει πότε οι αποδείξεις της ελέγχθηκαν τελευταία φορά· η επιστήμη κινείται, και οι παλιές αναφορές επανεπαληθεύονται πριν βασιστούν ξανά.
